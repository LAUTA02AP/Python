# Resolución de Ejercicios

En este directorio se encuentran las carpetas correspondientes a distintos conjuntos de ejercicios.

Cada carpeta incluye:

- **Enunciados** de los ejercicios planteados.
- Un archivo `.py` por cada ejercicio, donde se desarrolla su respectiva **resolución**.
- Todos los ejercicios están **importados y gestionados desde un archivo `main.py`**, el cual actúa como **menú de selección**, permitiendo al usuario elegir la actividad que desea ejecutar o visualizar.

Este enfoque permite una organización clara, modular y escalable para acceder a las distintas soluciones desde un solo punto de entrada.
